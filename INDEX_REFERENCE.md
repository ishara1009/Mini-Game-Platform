======================================================================
                      GAMES GUIDE
======================================================================

TIC TAC TOE:
  How to Play:
    - X and O take turns
    - Click a cell to place your mark
    - Get 3 in a row to win
    - If board fills, it's a draw
    - Winning squares turn gold
    - Click "Restart Game" to play again

  Winning Lines:
    - Horizontal (3 rows)
    - Vertical (3 columns)
    - Diagonal (2 diagonals)

MEMORY GAME:
  How to Play:
    - Click cards to flip and reveal emojis
    - Try to match pairs
    - All 8 pairs must be matched to win
    - Game tracks your move count
    - Matches can't be unflipped
    - Click "Play Again" to retry

  Difficulty:
    - Easier with memory practice
    - Pattern recognition helps
    - 8 different emojis = 16 cards total

======================================================================
                    TROUBLESHOOTING
======================================================================

"npm: command not found"
  → Install Node.js: https://nodejs.org/
  → Download LTS version
  → Restart terminal after installation

"Port 3000 already in use"
  → Use different port: PORT=3001 npm start

"Blank white page"
  → Check browser console (F12)
  → Clear cache: Ctrl+Shift+Delete
  → Restart dev server

"Styles not loading"
  → Delete node_modules folder
  → Run: npm install
  → Run: npm start

"Games not working"
  → Check browser console for errors
  → Verify all files exist
  → Restart development server
